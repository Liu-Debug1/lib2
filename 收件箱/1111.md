# STM32F103C8T6 外围模块引脚分配清单

## GPIOA

|  引脚  | 信号            | 连接模块            | 功能                 |  方向  |
| :--: | ------------- | --------------- | ------------------ | :--: |
| PA0  | KEY_MAIN_UP   | 主驾升窗按键          | 低电平有效，内部上拉         |  输入  |
| PA1  | KEY_MAIN_DOWN | 主驾降窗按键          | 低电平有效，内部上拉         |  输入  |
| PA2  | BT_TX         | HC-05 蓝牙模块 RX   | USART2 TX，9600bps  |  输出  |
| PA3  | BT_RX         | HC-05 蓝牙模块 TX   | USART2 RX，9600bps  |  输入  |
| PA6  | PWM_MAIN      | TB6612 PWMA     | TIM3 CH1，10kHz PWM | 复用推挽 |
| PA7  | PWM_PASS      | TB6612 PWMB     | TIM3 CH2，10kHz PWM | 复用推挽 |
| PA9  | VOICE_TX      | ASR-PRO 语音模块 RX | USART1 TX，9600bps  | 复用推挽 |
| PA10 | VOICE_RX      | ASR-PRO 语音模块 TX | USART1 RX，9600bps  | 上拉输入 |

> [!warning] PA1 / PA2 旧版遗留
> LED.c 中定义了 PA1 为 LED1、PA2 为 LED2（推挽输出），但当前主程序未调用 LED 函数。
> 原理图如不使用 LED 可忽略这两个引脚。

## GPIOB

| 引脚 | 信号 | 连接模块 | 功能 | 方向 |
|:----:|------|----------|------|:----:|
| PB0 | KEY_PASS_UP | 副驾升窗按键 | 低电平有效，内部上拉 | 输入 |
| PB1 | KEY_PASS_DOWN | 副驾降窗按键 | 低电平有效，内部上拉 | 输入 |
| PB3 | IR_TOP | 上置式红外对射 | 遮挡→高，双边沿中断 EXTI3 | 上拉输入 |
| PB4 | IR_BOTTOM | 下置式红外对射 | 遮挡→高，双边沿中断 EXTI4 | 上拉输入 |
| PB6 | I2C1_SCL | INA219 × 2 + OLED | 硬件 I2C 时钟线，100kHz | 复用开漏 |
| PB7 | I2C1_SDA | INA219 × 2 + OLED | 硬件 I2C 数据线，100kHz | 复用开漏 |
| PB8 | OLED_SCL | OLED SSD1306 | 软件 I2C 时钟（GPIO 模拟）| 开漏输出 |
| PB9 | OLED_SDA | OLED SSD1306 | 软件 I2C 数据（GPIO 模拟）| 开漏输出 |
| PB11 | DHT11_DATA | DHT11 温湿度传感器 | 单总线协议 | 双向 |
| PB12 | AIN1 | TB6612 主驾通道 | 主驾电机方向控制 H 桥输入 1 | 推挽输出 |
| PB13 | AIN2 | TB6612 主驾通道 | 主驾电机方向控制 H 桥输入 2 | 推挽输出 |
| PB14 | BIN1 | TB6612 副驾通道 | 副驾电机方向控制 H 桥输入 1 | 推挽输出 |
| PB15 | BIN2 | TB6612 副驾通道 | 副驾电机方向控制 H 桥输入 2 | 推挽输出 |

> [!warning] OLED 使用软件 I2C (PB8/PB9)，不与 INA219 共享硬件 I2C1 总线
> INA219 挂载在硬件 I2C1 (PB6/PB7)，OLED 用 GPIO 模拟 I2C (PB8/PB9)，两路物理独立。

## 内部资源（无外部引脚）

| 资源 | 用途 |
|------|------|
| TIM2 | 1ms 系统时基中断（g_SysTick 递增 + Key_Tick 按键扫描）|
| TIM3 | 10kHz PWM 输出（CH1=PA6 主驾，CH2=PA7 副驾）|
| I2C1 | 硬件 I2C（PB6/PB7），挂 INA219×2 |

## I2C 总线设备地址

| 设备 | 7-bit 地址 | 备注 |
|------|:---------:|------|
| INA219 主驾 | 0x40 | A0=GND, A1=GND |
| INA219 副驾 | 0x41 | A0=VS, A1=GND（需硬件确认） |
| OLED SSD1306 | 0x78 | 软件 I2C，不走硬件总线 |

## 电源与共地

```text
STM32 3.3V ──┬── TB6612 VCC (逻辑电源)
              ├── HC-05 VCC (3.3V，不可接 5V)
              ├── ASR-PRO VCC
              ├── DHT11 VCC (3.3V~5V)
              ├── OLED VCC (3.3V)
              ├── INA219×2 VCC (3.3V)
              └── 红外对射×2 VCC

外部电源 ──── TB6612 VM (电机电源，5~12V)
              所有模块 GND 必须共地
```
