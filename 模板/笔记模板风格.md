##### (3) 单调栈

**常用于解决**：
1. 正常片段
2. 找左边/右边第一个最大的数组
3. 对某一个数组进行单调增减的排列

- 正常片段
- 找左边/右边第一个最大的数组
- 对某一个数组进行单调增减的排列

```cpp
vector<int> ans(n, 0);
stack<int> st;  // 存还没被解决的下标

for (int i = 0; i < n; i++) {
    while (!st.empty() && nums[i] > nums[st.top()]) {
        int prevIndex = st.top();
        st.pop();

        ans[prevIndex] = i - prevIndex;  // 或者 ans[prevIndex] = nums[i]
    }

    st.push(i);
}
```
其中：`stack<int> st` 是用于存储位于原来数组的==下标==，下标顺序可以是原容器==单调递增/递减==的顺序
