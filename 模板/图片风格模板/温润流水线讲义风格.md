# 温润流水线讲义风格

![[模板/图片风格模板/温润流水线讲义风格-参考.png|900]]

## 一、风格定位

这是一种**温润、克制、讲义感很强的横向流程图风格**。

它不像商业海报那样强调视觉冲击，也不像科技风信息图那样堆叠光效和图标，而是像一页整理得很干净的课堂讲义：用留白、圆角框、细线箭头和一个橙色重点节点，把复杂流程讲清楚。

## 二、适用场景

1. 解释一个流程
   - 例如：AI 工作流、论文实验流程、嵌入式开发流程、线控底盘控制链路。

2. 拆解一个方法论
   - 例如：学习闭环、项目推进路径、科研训练路径、产品开发步骤。

3. 做知识卡片或朋友圈配图
   - 适合表达“清楚、沉稳、有知识感”的内容，不适合做强营销海报。

## 三、视觉规则

| 维度 | 规则 |
|---|---|
| 画布 | 横向长图，比例接近 `1260 x 470` |
| 背景 | 暖米白、纸张感、低对比度 |
| 主结构 | 4 个左右均分的圆角矩形流程节点 |
| 连接 | 节点之间使用细灰色箭头 |
| 强调 | 只强调 1 个关键节点，使用浅橙底和橙色描边 |
| 标签 | 每个流程节点下方放一个胶囊形说明标签 |
| 文字 | 现代中文无衬线，灰色为主，橙色只用于重点 |
| 情绪 | 温和、理性、克制、像课程讲义 |

## 四、配色参考

| 用途 | 建议颜色 |
|---|---|
| 背景 | `#f7f3ea` / `#f5f1e8` |
| 普通文字 | `#6f6d68` |
| 次级文字 | `#9a9287` |
| 普通边框 | `#c8c3b8` |
| 箭头 | `#aaa69e` |
| 强调橙 | `#d35a35` |
| 强调浅底 | `#f5ddd0` |

## 五、固定版式

```text
                  顶部标题

┌──────────┐  →  ┌──────────┐  →  ┌──────────┐  →  ┌──────────┐
│  节点 1   │     │  节点 2   │     │  节点 3   │     │  节点 4   │
│  副标题   │     │  副标题   │     │  副标题   │     │  副标题   │
└──────────┘     └──────────┘     └──────────┘     └──────────┘
  胶囊标签         胶囊标签         胶囊标签         胶囊标签

                  底部总结句
```

> [!note]
> 默认第 3 个节点作为强调节点；如果主题里有更关键的阶段，可以改为第 2 或第 4 个节点，但整张图只保留一个强调节点。

## 六、可替换变量

| 变量 | 说明 |
|---|---|
| `{主题}` | 这张图要解释的核心主题 |
| `{顶部标题}` | 一句话概括整张图 |
| `{节点1-4}` | 四个流程阶段 |
| `{副标题1-4}` | 每个节点的短解释，建议 2-5 个字 |
| `{标签1-4}` | 节点下方的动作说明 |
| `{强调节点}` | 默认第 3 个 |
| `{底部总结}` | 用一句话收束整张图 |

## 八、调用方式

以后可以直接说：

> 用“温润流水线讲义风格”生成一张关于 `{主题}` 的流程图。

如果没有特别说明，默认使用：

1. 4 个流程节点。
2. 第 3 个节点作为强调节点。
3. 顶部一句标题，底部一句总结。
4. 米白背景、灰色主体、单一橙色重点。
