# 秘书团使用指南

> 由 **总秘书大人** 统一调度，复盘官 + 问题官并行工作。

## 你会这样对我说

| 你说 | 总秘书大人做什么 |
|------|-----------------|
| "复盘" 或 "考试" | 复盘官拉到期错题（全做） → 问题官同步后台出新题 → 错题考完接新题 |
| "出题" | 问题官独立手工出题，写入题库 |
| "批改" | 知识官批改答卷，更新状态库 |

## 每日流程（一句话）

说"复盘" → 错题全部过一遍 → 新题自动衔接，一天的量就完成了。

## 新题数量

```
默认新题基准 = 8（可调）
错题量 M → 新题量 = 8 - floor(M/5)，最少 3
M=4  → 新题 8    M=5  → 新题 7    M=11 → 新题 6
```

## 1-3-7 错题复习（自动）

答错 → 隔天复习 → 正确 → 第3天 → 正确 → 第7天 → 正确 → 删除。任意错误回退隔天。
