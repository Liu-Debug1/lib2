## 三、控制算法架构

```
             ┌─────────────┐
 领航车 ──────→ 加速度/速度 ──→ 车队传播
             └─────────────┘
                    │
        ┌───────────┼───────────┐
        ▼           ▼           ▼
   ┌─────────┐ ┌─────────┐ ┌─────────┐
   │ 位置跟踪 │ │ 速度跟踪 │ │ 间隙控制 │
   └────┬────┘ └────┬────┘ └────┬────┘
        │           │           │
        └───────────┼───────────┘
                    ▼
        ┌─────────────────────┐
        │   耦合滑模控制器      │  ← 有限时间收敛
        │   S = e_v + β·e_p   │
        └─────────┬───────────┘
                  │
                  ▼
        ┌─────────────────────┐
        │   扰动观测器 (DO)    │  ← 估计失配扰动
        │   d̂ → 前馈补偿       │
        └─────────┬───────────┘
                  │
                  ▼
        ┌─────────────────────┐
        │   输入饱和限制       │  ← am ~ aM 限幅
        └─────────┬───────────┘
                  │
                  ▼
           车辆动力学 (三阶模型)
```

### 核心公式

**车辆模型：**

```
ṗ_i = v_i
v̇_i = a_i
ȧ_i = -1/τ · a_i + 1/τ · u_i + d_i
```

其中 τ 为执行器延迟，d_i 为失配扰动。

**耦合滑模面：**

```
s_i = e_{v,i} + β · e_{p,i}
```

β 为耦合参数，将位置误差和速度误差融合到同一个滑模面。

**控制律（有限时间）：**

```
u_i = −k₁ · |s_i|^{α} · sign(s_i) − k₂ · ∫ sign(s_i) − d̂_i
```

## 四、四种控制模式对比

代码通过 `flag` 开关自由组合，可直接对比四种方案：

| 模式 | 控制器 | 扰动观测器 | flag 配置 |
| ---- | ------ | ---------- | --------- |
| 基线 | 线性反馈 | 无 | `flagFinitefCtrl=0, flagDO=0` |
| 增强 A | 线性反馈 | 有限时间 DO | `flagFinitefCtrl=0, flagDO=1` |
| 增强 B | 有限时间滑模 | 无 | `flagFinitefCtrl=1, flagDO=0` |
| **论文方案** | **有限时间滑模** | **有限时间 DO** | `flagFinitefCtrl=1, flagDO=1` |

## 五、支持的通信拓扑（5 种）

```
PF   (IndexTopo=1):  尾车 ← 前车 ← 前车 ← 领航
PLF  (IndexTopo=2):  每辆车 ← 前车 + 领航车
TPF  (IndexTopo=3):  尾车 ← 前两车
TPLF (IndexTopo=4):  每辆车 ← 前两车 + 领航车
MF   (IndexTopo=5):  多前车（中间车作微型领航）
```

## 六、仿真场景

| 参数 | 默认值 | 说明 |
| ---- | ------ | ---- |
| 车辆数 Nveh | 6 | 1 领航 + 5 跟随 |
| 初始速度 v0 | 16 m/s | 高速公路场景 |
| 时距 h | 0.5 s | 跟车时距 |
| 最小安全距离 ddes | 8 m | 静止间隙 |
| 执行器延迟 τ | 0.3 s | 一阶惯性环节 |
| 仿真步长 Tstep | 0.01 s | |
| 仿真步数 Nstep | 6500 | ~65 s 仿真时间 |
| 扰动注入时刻 | t=1020 s | 从该时刻起注入失配扰动 |
| 扰动幅值 | dd1=0.3, dd2=1.5 | |

### 可注入的扰动类型

| 开关 | 含义 | 实际场景对应 |
| ---- | ---- | ------------ |
| `flagmisdis=1` | 失配扰动 | 道路坡度、风阻、载荷变化 |
| `flagParaMismatch=1` | 参数不匹配 | 不同车辆质量/轮胎差异 |
| `flagCtrlSat=1` | 输入饱和 | 油门/制动有物理上限 |

## 九、复现步骤

1. MATLAB 打开 `sim4_1.m`
2. F5 运行，默认配置跑通（~30 秒）
3. 改 `IndexTopo` 对比五种拓扑
4. 改 `flagDO` / `flagFinitefCtrl` 对比四种控制模式
5. 将 `v0`、`ddes`、`h`、`tau` 换成压路机参数

## 十一、评价

### 优势

- **代码极小**：单文件 700 行，无复杂依赖，十分钟跑通
- **算法完整**：滑模 + DO + 多拓扑 + 输入饱和，一套全有
- **论文对照**：每个变量对应论文公式，适合学习
- **对比实验现成**：flag 切四种模式，论文对比图直接出
- **异构支持**：每辆车可设不同参数，贴近真实车队

### 不足

- 车辆模型简化（三阶线性），无轮胎/悬架动力学
- 无横向控制，仅纵向跟车
- 注释中文编码乱码（GBK/UTF-8 问题）
- 无 Simulink 可视化，纯数据曲线
- 社区活跃度低（35 star，单文件，无 issue 讨论）

### 适用度总评

| 维度 | 评分 | 说明 |
| ---- | :--: | ---- |
| 论文复现 | ★★★★★ | 代码与论文一一对应 |
| 算法学习 | ★★★★☆ | 滑模+DO 的完整教学级实现 |
| 压路机直接移植 | ★★★☆☆ | 三阶模型可用，需改参数 |
| 工程部署 | ★★☆☆☆ | MATLAB 仿真，非嵌入式代码 |

**结论：最适合作为 CACC 编队控制的算法入门和论文对比基线。代码量小、环境简单、论文支撑强。压路机场景需改参数和车辆模型，但控制器结构可直接复用。**
