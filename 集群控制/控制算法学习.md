# 集群控制算法学习资料

## 学习路线

```
代数图论 → 一阶/二阶一致性 → 编队控制 → 分布式MPC → 事件触发进阶
```

## 一、一致性控制（Consensus）

**推荐教材**：Nagahara 等 *Control of Multi-agent Systems: Theory and Simulations with Python* (Springer 2024)，含图论基础、一致性协议及 Python 代码。

**B站资源**：

| UP主     | 内容                    | 难度  |
| ------- | --------------------- | --- |
| 桂花米酒酿   | 代数图论 → 单积分器 → 双积分器一致性 | 入门  |
| 辣酱超人    | 论文讲解，编队控制基础           | 进阶  |
| uuid975 | MATLAB 仿真实操           | 实战  |
| AjenZzz | 6分钟快速入门一致性            | 速览  |

## 二、编队控制（Formation Control）

三大经典方法：

| 方法                         | 原理       | 优缺点          |
| -------------------------- | -------- | ------------ |
| Leader-Follower            | 领航者带跟随者  | 简单，领航者单点故障   |
| Virtual Structure          | 提供期望虚拟轨迹 | 无单点风险，需全局信息  |
| Artificial Potential Field | 引力+斥力场   | 天然避障，局部极小值问题 |

**GitHub 项目**：
- `MosesEbere/multi-robot-consensus` — ROS+Python 多机器人一致性
- `Sang-Buster/Formation-Control` — Python+MATLAB 编队控制
- `wang-chen/formation` — ROS 多无人机编队

**B站**：Joymoss 的"一致性+APF编队避障"系列

## 四、事件触发控制

**核心思想**：用"按需通信"替代周期采样，通信量可减少 80-98%。

**B站**：辣酱超人/既然香遇的"多智能体事件触发控制论文讲解"

**CSDN**：多套 MATLAB 仿真代码（静态触发、动态触发、固定时间触发等变体）
