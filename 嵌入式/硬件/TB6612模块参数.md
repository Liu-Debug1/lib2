# TB6612 成品模块 PCB 设计参数

## 1. 市面常见版本

### 1.1 DFRobot / 淘宝通用微型模块（最常见）

| 项目 | 规格 |
|:-----|:-----|
| **PCB 尺寸** | **20mm × 19.5mm** |
| **引脚排列** | 两侧各 6 根排针，共 12 引脚 |
| **引脚间距** | **2.54mm**（标准 0.1" 排针） |
| **安装孔** | 无（仅靠排针/插座固定） |
| **PCB 颜色** | 红 / 蓝 |

> 淘宝销量最大，价格 5~10 元。本毕设项目使用此版本的可能性最大。

### 1.2 其他常见版本

| 版本 | PCB 尺寸 | 安装孔 | 特点 |
|:-----|:---------|:-------|:-----|
| **Pololu Carrier** | 15.2 × 20.3mm | 无 | 1×16 排针，极小体积 |
| **SparkFun** | 20.3 × 20.3mm | 无 | 带滤波电容 |
| **SunFounder** | 42 × 32mm | 4 孔 × 3mm | 带端子排 |
| **光电隔离款** | 50.2 × 48.2mm | 4 孔 × 3mm（孔距 40×38mm） | 光耦隔离、防反接、板载 5V LDO |

### 1.3 通用参数

| 参数 | 值 |
|:-----|:---|
| **引脚间距** | **2.54mm**（所有版本统一） |
| **安装孔径** | **3mm**（有安装孔的版本） |
| **排针类型** | 标准 0.1" 排针，可直插面包板/排母 |

## 2. TB6612FNG 芯片本体参数

| 参数 | 值 |
|:-----|:---|
| 制造商 | Toshiba（东芝） |
| 结构 | 双路 H 桥，LD MOS |
| 封装 | SSOP-24 |
| 芯片尺寸 | 8.3mm × 5.6mm |
| 引脚间距 | 0.65mm |

### 电气参数

| 参数 | 值 |
|:-----|:---|
| 电机电源 VM | 2.5V ~ 13.5V（最大 15V） |
| 逻辑电源 VCC | 2.7V ~ 5.5V |
| 持续输出电流 | 1.2A（每通道平均） |
| 峰值输出电流 | 3.2A（单脉冲） |
| 导通电阻 | 0.5Ω（typ，VM ≥ 5V） |
| 最大 PWM 频率 | 100kHz |
| 控制模式 | CW / CCW / 短制动 / 停止 |
| 保护功能 | 过热关断 + 低压检测 |

## 3. 引脚功能表

| 引脚 | 类型 | 功能 |
|:-----|:-----|:-----|
| **VM** | 电源 | 电机电源（2.5~13.5V） |
| **VCC** | 电源 | 逻辑电源（2.7~5.5V） |
| **GND** | 电源 | 地，必须与 MCU 共地 |
| **STBY** | 输入 | 待机控制（高电平 = 正常工作） |
| **AIN1** | 输入 | A 通道方向控制 1 |
| **AIN2** | 输入 | A 通道方向控制 2 |
| **PWMA** | 输入 | A 通道 PWM 速度控制 |
| **BIN1** | 输入 | B 通道方向控制 1 |
| **BIN2** | 输入 | B 通道方向控制 2 |
| **PWMB** | 输入 | B 通道 PWM 速度控制 |
| **AO1** | 输出 | A 通道电机输出 1 |
| **AO2** | 输出 | A 通道电机输出 2 |
| **BO1** | 输出 | B 通道电机输出 1 |
| **BO2** | 输出 | B 通道电机输出 2 |

### 控制真值表（以 A 通道为例）

| AIN1 | AIN2 | PWMA | 电机状态 |
|:-----|:-----|:-----|:---------|
| 1 | 0 | PWM | 正转（调速） |
| 0 | 1 | PWM | 反转（调速） |
| 1 | 1 | — | 制动（短制动） |
| 0 | 0 | — | 停止（滑行） |

## 5. 本毕设项目接线

| STM32 引脚 | TB6612 引脚 | 功能 |
|:-----------|:------------|:-----|
| PA6 (TIM3_CH1) | PWMA | 主驾 PWM 调速 |

| PB13 | AIN2 | 主驾方向控制 |
