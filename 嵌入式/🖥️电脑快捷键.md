## 电脑快捷键

### 通用编辑

| 快捷键 | 功能 |
|:------|:-----|
| `Ctrl + A` | 全选 |
| `Ctrl + C` | 复制 |
| `Ctrl + X` | 剪切 |
| `Ctrl + V` | 粘贴 |
| `Ctrl + Z` | 撤销上一步操作 |
| `Ctrl + S` | 保存 |

### 代码编辑

| 快捷键           | 功能            |
| :------------ | :------------ |
| `Ctrl + D`    | 复制光标行到下一行     |
| `Ctrl + P`    | 显示调用方法参数类型与个数 |
| `Alt + Enter` | 自动搜索并导包       |

### 窗口管理

| 快捷键         | 功能                              |
| :---------- | :------------------------------ |
| `Alt + Tab` | 切屏（长按，松开返回上一个窗口，继续按 Tab 切换其他窗口） |
| `Win + D`   | 显示桌面                            |
| `Win + E`   | 打开文件资源管理器                       |
| `Win + L`   | 锁定电脑                            |
| `Win + ←/→` | 窗口分屏（左/右半屏）                     |
| `Win + ↑/↓` | 窗口最大化/还原                        |

### Windows 终端

| 快捷键                  | 功能                        |
| :------------------- | :------------------------ |
| `鼠标右键`               | 粘贴（CMD/PowerShell）        |
| `Ctrl + Shift + V`   | 粘贴（Windows Terminal）      |
| `Ctrl + Shift + C`   | 复制（Windows Terminal）      |
| `Ctrl + Insert`      | 复制                        |
| `Shift + Insert`     | 粘贴                        |
| `F7`                 | CMD 命令行历史列表               |
| `Esc`                | 清空当前行（部分交互程序会拦截，如 Claude Code） |
| `Ctrl + Backspace`   | 删除光标前一个单词（不受应用拦截）      |
| `Ctrl + ←/→`         | 按单词左右跳光标                |
| `↑ / ↓`              | 切换历史命令                    |
| `Tab`                | 自动补全路径/命令                 |
| `Ctrl + C`           | 终止当前命令                    |
| `Ctrl + Shift + T`   | 新建标签页（Windows Terminal）   |
| `Ctrl + Shift + W`   | 关闭标签页（Windows Terminal）   |
| `Alt + F4`           | 关闭终端窗口                    |
| `Ctrl + Shift + +/-` | 放大/缩小字号（Windows Terminal） |

### Linux 终端

| 快捷键 | 功能 |
|:------|:-----|
| `鼠标右键` | 粘贴 |
| `Ctrl + Shift + V` | 粘贴 |
| `Ctrl + Shift + C` | 复制 |
| `Ctrl + Insert` | 复制 |
| `Shift + Insert` | 粘贴 |
| `↑ / ↓` | 切换历史命令 |
| `Tab` | 自动补全路径/命令 |
| `Ctrl + C` | 终止当前命令 |
| `Ctrl + U` | 清空当前行 |
| `Ctrl + K` | 删除光标到行尾 |
| `Ctrl + W` | 删除光标前一个单词 |
| `Ctrl + A` | 光标跳到行首 |
| `Ctrl + E` | 光标跳到行尾 |
| `Ctrl + L` | 清屏 |
| `Ctrl + D` | 退出终端 / 发送 EOF |
| `Ctrl + R` | 搜索历史命令 |
