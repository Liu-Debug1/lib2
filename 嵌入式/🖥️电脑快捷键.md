## 电脑快捷键

### 通用编辑

| 快捷键 | 功能 |
|:------|:-----|
| `Ctrl + A` | 全选 |
| `Ctrl + C` | 复制 |
| `Ctrl + X` | 剪切 |
| `Ctrl + V` | 粘贴 |
| `Ctrl + Z` | 撤销上一步操作 |
| `Ctrl + S` | 保存 |

### 代码编辑

| 快捷键           | 功能            |
| :------------ | :------------ |
| `Ctrl + D`    | 复制光标行到下一行     |
| `Ctrl + P`    | 显示调用方法参数类型与个数 |
| `Alt + Enter` | 自动搜索并导包       |

### 窗口管理

| 快捷键         | 功能                              |
| :---------- | :------------------------------ |
| `Alt + Tab` | 切屏（长按，松开返回上一个窗口，继续按 Tab 切换其他窗口） |
| `Win + D`   | 显示桌面                            |
| `Win + E`   | 打开文件资源管理器                       |
| `Win + L`   | 锁定电脑                            |
| `Win + ←/→` | 窗口分屏（左/右半屏）                     |
| `Win + ↑/↓` | 窗口最大化/还原                        |
