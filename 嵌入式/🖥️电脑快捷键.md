# Windows

## 一、先区分三层

Windows 终端相关快捷键不要混在一起记，实际要分三层：

| 层级 | 负责什么 | 典型例子 |
|:----|:----|:----|
| 终端模拟器 | 窗口、标签页、分屏、复制粘贴 | Windows Terminal |
| Shell 行编辑 | 光标移动、删除、历史搜索、自动补全 | PowerShell 的 PSReadLine、CMD 行编辑 |
| 命令本身 | 文件、进程、网络、系统信息操作 | `dir`、`Get-ChildItem`、`tasklist` |

> [!NOTE]
> `Ctrl + C` 在普通文本编辑器里是复制，但在终端 Shell 里通常是中断前台程序。Windows Terminal 复制一般用 `Ctrl + Shift + C`，粘贴一般用 `Ctrl + Shift + V` 或 `Shift + Insert`。

## 四、PowerShell

### 4.1 PowerShell 行编辑（PSReadLine）

PowerShell 的输入行由 PSReadLine 处理。Windows 上默认偏 Windows 风格。可以用下面命令查看当前绑定：

```powershell
Get-PSReadLineOption
Get-PSReadLineKeyHandler
Get-PSReadLineKeyHandler -Chord Ctrl+r
```

| 快捷键 | 功能 |
|:------|:-----|
| `←/→` | 左右移动一个字符 |
| `Ctrl + ←/→` | 按单词左右移动 |
| `Home / End` | 移动到行首/行尾 |
| `Backspace` | 删除光标前一个字符 |
| `Delete` | 删除光标处一个字符 |
| `Ctrl + Backspace` | 删除光标前一个单词 |
| `Ctrl + Delete` | 删除光标后一个单词 |
| `Ctrl + Home` | 删除光标到行首 |
| `Ctrl + End` | 删除光标到行尾 |
| `Esc` | 清空当前输入行 |
| `Tab` | 补全命令、参数或路径；重复按可轮换候选 |
| `Ctrl + Space` | 菜单式补全 |
| `Shift + Enter` | 输入多行命令而不立即执行 |

> [!warning] 不要死记 `Ctrl + A/E`
> 在 Bash/Readline 中，`Ctrl + A` 是行首、`Ctrl + E` 是行尾；但 PowerShell 的默认 Windows 编辑模式更推荐用 `Home / End`。如果你把 PowerShell 改成 Emacs 模式，`Ctrl + A/E` 才更像 Bash。

## 六、Windows 易错点

- `Ctrl + C`
  - 文本编辑器：复制
  - 终端 Shell：中断前台程序
  - 终端复制：通常用 `Ctrl + Shift + C`
- Windows 路径用反斜杠 `\`
- CMD 跨盘 `cd` 要加 `/d`，例如 `cd /d D:\work`
- PowerShell 的 `ls`、`rm`、`cat` 很多是别名，不完全等于 Linux 命令
- PowerShell 管道传对象，CMD 管道传文本
- `rd /s /q`、`robocopy /mir` 都有破坏性，执行前确认路径
- `wmic` 在新 Windows 中不推荐作为首选，优先学 PowerShell `Get-CimInstance`

# Linux

## 一、先区分三层

Linux 终端相关快捷键同样要分三层：

| 层级 | 负责什么 | 典型例子 |
|:----|:----|:----|
| 终端模拟器 | 窗口、标签页、复制粘贴 | GNOME Terminal、Konsole、xterm |
| Shell 行编辑 | 光标移动、删除、历史搜索、自动补全 | Bash 的 GNU Readline |
| 命令本身 | 文件、进程、网络、系统信息操作 | `ls`、`grep`、`ps`、`ip` |

> [!NOTE]
> Linux 下很多“终端快捷键”其实是 Bash/Readline 的行编辑快捷键。如果换成 Zsh、Fish 或启用 Vi 模式，部分快捷键会变化。

## 四、Linux 易错点

- Linux 路径用斜杠 `/`
- Linux 管道主要传文本
- `rm -rf` 有破坏性，执行前确认路径
- `Ctrl + S` 在 Linux 终端可能暂停输出，误按后用 `Ctrl + Q` 恢复
- `ifconfig`、`netstat` 在新 Linux 发行版中可能未默认安装，优先学 `ip` 和 `ss`
- Bash/Readline 快捷键依赖 Shell 和编辑模式，换成 Zsh、Fish 或 Vi 模式后可能不同
