### 一、通用快捷键

| 功能       | Windows/Linux        | macOS         | 说明                      |
| -------- | -------------------- | ------------- | ----------------------- |
| 打开命令面板   | `Ctrl+P`             | `Cmd+P`       | 最核心的万能入口，几乎任何操作都能搜到     |
| 快速切换文件   | `Ctrl+O`             | `Cmd+O`       | 快速跳转到其他笔记，比鼠标点左边文件列表快得多 |
| 搜索当前库    | `Ctrl+Shift+F`       | `Cmd+Shift+F` | 全文搜索所有笔记                |
| 打开/切换侧边栏 | `Ctrl+Shift+E`（文件列表） | 同左            | 开关左侧文件管理器               |
| 切换右侧面板   | `Ctrl+Shift+Right`   | 同左            | 显示大纲、反链面板等              |
| 打开设置     | `Ctrl+,`             | `Cmd+,`       | 进入设置页面                  |

--- 
### 二、文档内部

| 功能          | 快捷键                            | 说明                            |
| ----------- | ------------------------------ | ----------------------------- |
| **加粗**      | `Ctrl+B` / `Cmd+B`             | 选中文字加粗                        |
| **斜体**      | `Ctrl+I` / `Cmd+I`             | 选中文字变为斜体                      |
| **高亮**      | `Ctrl+Shift+H`/Alt+H           | 给文字加上高亮                       |
| 插入内链        | `Ctrl+K` / `Cmd+K`             | 弹出链接输入框，快速链接到其他笔记或URL         |
| 缩进/反缩进      | `Tab` / `Shift+Tab`            | 在列表、引用块、代码块中很有用               |
| 上移/下移当前行    | `Alt+↑` / `Alt+↓`              | 整行移动，不必用鼠标拖拽                  |
| 复制当前行       | `Ctrl+D` / `Cmd+D`             | 复制整行到下方，适合快速重复                |
| **删除当前行**   | `Ctrl+Shift+K` / `Cmd+Shift+K` | 快速删除整行，无需选中                   |
| 插入当前日期      | `Ctrl+Shift+;`                 | 插入 `YYYY-MM-DD` 格式日期          |
| **注释/取消注释** | `Ctrl+/`                       | 在代码块或 YAML Front Matter 中切换注释 |
| 切换复选框       | `Ctrl+Enter`                   | 在任务列表行上切换 `- [ ]` ↔ `- [x]`   |
| 撤销/重做       | `Ctrl+Z` / `Ctrl+Y`            | 标准操作                          |
### 三、视图与导航
| 功能          | 快捷键                         | 说明                    |
| ----------- | --------------------------- | --------------------- |
| 返回/前进（浏览历史） | `Ctrl+Alt+←` / `Ctrl+Alt+→` | 像浏览器的后退/前进，非常实用       |
| 打开链接/笔记     | `Ctrl+点击`                   | 按住 `Ctrl` 点击内链，在新窗格打开 |
| 打开当前笔记的预览   | `Ctrl+E` / `Cmd+E`          | 切换编辑/预览模式             |
| 聚焦到编辑区      | `Ctrl+Shift+L`              | 当焦点在其他面板时，一键回到编辑器     |
| 展开/折叠标题     | `Ctrl+Shift+↑/↓`            | 在大纲或文件列表中折叠标题层级       |
| 在新窗格中打开     | `Ctrl+Shift+Enter`          | 在分割视图中打开当前笔记          |

### 四 、Callout（标注块）
| 类型                           | 显示效果   |
| ---------------------------- | ------ |
| `[!note]`                    | 蓝色笔记   |
| `[!tip]` / `[!hint]`         | 绿色提示   |
| `[!warning]` / `[!caution]`  | 黄色警告   |
| `[!danger]`                  | 红色危险   |
| `[!info]`                    | 蓝色信息   |
| `[!abstract]` / `[!summary]` | 灰色摘要   |
| `[!example]`                 | 紫色示例   |
| `[!quote]` / `[!cite]`       | 灰色引用   |
| `[!success]` / `[!done]`     | 绿色成功   |
| `[!failure]` / `[!fail]`     | 红色失败   |
| `[!bug]`                     | 红色 Bug |
| `[!question]` / `[!help]`    | 绿色疑问   |
