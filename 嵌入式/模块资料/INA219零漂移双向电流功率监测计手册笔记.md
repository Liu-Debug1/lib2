# INA219 零漂移双向电流/功率监测计

> [!info] 适用场景
> 本芯片符合工业/汽车级精度要求，常用于汽车电子电流监测、电源管理、电池充电器、线控执行器电流反馈等场景
> - 接口：I2C / SMBus 兼容
> - 关键特性：可编程校准、12位ADC、PGA、总线电压测量 0~26V、16个可选从机地址

## 1. 特性
- 总线电压测量范围：0V 至 26V
- 分流电压测量范围：±40mV、±80mV、±160mV、±320mV（通过 PGA 设置）
- 可编程校准寄存器（Calibration Register），实现直接读电流（单位安培）和功率（单位瓦）
- 16 个可编程地址（通过 A0、A1 引脚）
- 零漂移架构，精度高（INA219B 最大 0.5% 全温范围）
- 滤波选项（分辨率与平均次数可编程）
- 封装：SOT-23-8、SOIC-8

## 3. I2C 从机地址
A0、A1 引脚电平组合决定 7 位地址，上电后每次通信都会采样引脚状态。

| A1  | A0  | 从机地址（7位）  |
| :-- | :-- | :-------- |
| GND | GND | `1000000` |
| GND | VS+ | `1000001` |
| GND | SDA | `1000010` |
| GND | SCL | `1000011` |
| VS+ | GND | `1000100` |
| VS+ | VS+ | `1000101` |
| VS+ | SDA | `1000110` |
| VS+ | SCL | `1000111` |
| SDA | GND | `1001000` |
| SDA | VS+ | `1001001` |
| SDA | SDA | `1001010` |
| SDA | SCL | `1001011` |
| SCL | GND | `1001100` |
| SCL | VS+ | `1001101` |
| SCL | SDA | `1001110` |
| SCL | SCL | `1001111` |

> [!tip] 常用地址示例
> 若 A0=GND, A1=GND，则从机地址为 `0x40`（写操作）和 `0x41`（读操作），无需额外硬件配置即可使用。

### 4.3 I2C 写时序（参考官方手册 Figure 15）
```text
[开始] -> [从机地址+W] -> ACK -> [寄存器指针] -> ACK -> [数据高字节] -> ACK -> [数据低字节] -> ACK -> [停止]
```
> [!note] 字节序要求
> 所有数据字节均为 **高位先行（MSB first）**

### 4.4 I2C 读时序（参考官方手册 Figure 16）
```text
[开始] -> [从机地址+W] -> ACK -> [寄存器指针] -> ACK 
    -> [重复开始] -> [从机地址+R] -> ACK -> [数据高字节] -> ACK -> [数据低字节] -> NACK -> [停止]
```
> [!warning] 读操作注意事项
> 指定地址读必须使用**重复开始**信号，不得先发送停止信号再启动读操作，否则寄存器指针可能被复位导致读取错误。

## 5. 寄存器映射
| 地址 (Hex) | 名称 | 作用 | 默认值 | 访问权限 |
|:---|:---|:---|:---|:---|
| 00 | Configuration | 配置总线电压量程、PGA、ADC分辨率、模式等 | `0x399F` | R/W |
| 01 | Shunt Voltage | 分流电压测量值（数字量） | — | R |
| 02 | Bus Voltage | 总线电压测量值（数字量） | — | R |
| 03 | Power | 功率值（数字量） | `0x0000` | R |
| 04 | Current | 电流值（数字量） | `0x0000` | R |
| 05 | Calibration | 校准值，决定电流/功率的 LSB 精度 | `0x0000` | R/W |

> [!warning] 初始化要求
> 上电后电流和功率寄存器默认值为零，直到配置寄存器 (00h) 和校准寄存器 (05h) 被正确写入后才会输出有效值。

## 6. 配置寄存器 (0x00) 详解
默认值：`0x399F`
位分布定义：
```
bit15: RST    (系统复位位)
bit14: —      (保留位，必须写0)
bit13: BRNG   (总线电压量程选择)
bit12: PG1    (PGA增益高位)
bit11: PG0    (PGA增益低位)
bit10: BADC4  (总线ADC配置位4)
bit9:  BADC3  (总线ADC配置位3)
bit8:  BADC2  (总线ADC配置位2)
bit7:  BADC1  (总线ADC配置位1)
bit6:  SADC4  (分流ADC配置位4)
bit5:  SADC3  (分流ADC配置位3)
bit4:  SADC2  (分流ADC配置位2)
bit3:  SADC1  (分流ADC配置位1)
bit2:  MODE2  (工作模式位2)
bit1:  MODE1  (工作模式位1)
bit0:  MODE0  (工作模式位0)
```

### 6.1 BRNG (bit13) 总线电压量程
| BRNG | 量程 | LSB精度 |
|:---|:---|:---|
| 0 | 16V FSR | 4 mV |
| 1 | **32V FSR** (默认) | 4 mV |

### 6.2 PG (bit12:11) 可编程增益
| PG1 | PG0 | 增益 | 满量程范围 |
|:---|:---|:---|:---|
| 0 | 0 | /1 | ±40 mV |
| 0 | 1 | /2 | ±80 mV |
| 1 | 0 | /4 | ±160 mV |
| 1 | 1 | **/8** (默认) | ±320 mV |

> [!tip] PGA选择依据
> 确保最大分流电压 ($I_{max} \times R_{shunt}$) 不超过选定范围，兼顾测量精度和量程。

### 6.3 BADC (bit10:7) 和 SADC (bit6:3) ADC 分辨率/平均配置
| bit3-6 (SADC) 或 bit7-10 (BADC) | 分辨率/平均次数 | 转换时间 |
|:---|:---|:---|
| `0 X 0 0` | 9-bit | 84 μs |
| `0 X 0 1` | 10-bit | 148 μs |
| `0 X 1 0` | 11-bit | 276 μs |
| `0 X 1 1` | 12-bit | **532 μs** (默认) |
| `1 0 0 0` | 12-bit | 532 μs |
| `1 0 0 1` | 2 样本平均 | 1.06 ms |
| `1 0 1 0` | 4 样本平均 | 2.13 ms |
| `1 0 1 1` | 8 样本平均 | 4.26 ms |
| `1 1 0 0` | 16 样本平均 | 8.51 ms |
| `1 1 0 1` | 32 样本平均 | 17.02 ms |
| `1 1 1 0` | 64 样本平均 | 34.05 ms |
| `1 1 1 1` | 128 样本平均 | 68.10 ms |

> 注：X为任意值，为保持代码一致性建议置0。若无特殊噪声抑制要求，推荐选择`0011`（12-bit，532μs转换时间）兼顾精度和响应速度。

### 6.4 MODE (bit2:0) 工作模式
| MODE2 | MODE1 | MODE0 | 模式 |
|:---|:---|:---|:---|
| 0 | 0 | 0 | 掉电模式 |
| 0 | 0 | 1 | 分流电压，触发测量 |
| 0 | 1 | 0 | 总线电压，触发测量 |
| 0 | 1 | 1 | 分流和总线，触发测量 |
| 1 | 0 | 0 | ADC 禁用 |
| 1 | 0 | 1 | 分流电压，连续测量 |
| 1 | 1 | 0 | 总线电压，连续测量 |
| 1 | 1 | 1 | **分流和总线连续测量** (默认) |

> 常用配置：`111` 连续测量分流和总线电压，适合实时监测场景。

### 6.5 RST (bit15) 复位位
置1会触发系统复位，所有寄存器恢复默认值，该位硬件自动清零。

## 7. 数据输出寄存器
### 7.1 分流电压寄存器 (0x01)
- 存储最新分流电压（$V_{SHUNT}$）
- 分辨率：10 μV/LSB（所有PGA设置下固定不变）
- 数据格式：二进制补码（有符号整数，支持双向电流测量）
- 满量程对应十进制值：
  `/1: ±40mV → ±4000`
  `/2: ±80mV → ±8000`
  `/4: ±160mV → ±16000`
  `/8: ±320mV → ±32000`

**转换公式**：
$$V_{SHUNT} (mV) = 寄存器值 \times 0.01$$

### 7.2 总线电压寄存器 (0x02)
- 存储最新总线电压（$V_{BUS}$）
- 分辨率：4 mV/LSB
- 数据格式：右对齐，bit15~bit3为有效值，bit2~bit0为状态标志
- 读取后需要**右移3位**再乘以4mV得到实际电压值

**转换公式**：
$$V_{BUS}= (寄存器值 >> 3) \times 4 （单位mV）$$ 

**状态位定义**：
- bit1：CNVR (转换就绪)，1 = 新测量数据可用
- bit0：OVF (算术溢出)，1 = 功率或电流计算溢出，需调整校准参数

### 7.3 功率寄存器 (0x03)
- 内部自动计算： $POWER = (CURRENT \times BUS\_VOLTAGE) / 5000$
- 分辨率：$20 \times Current\_LSB$ (W/bit)
- 必须先写入校准寄存器才有有效值

### 7.4 电流寄存器 (0x04)
- 内部自动计算：$CURRENT = (SHUNT\_VOLTAGE \times CALIBRATION) / 4096$
- 分辨率：$Current\_LSB$ (A/bit)
- 有符号整数（支持双向电流测量）

## 8. 校准寄存器 (0x05) 及计算流程
> [!important] 核心配置
> 本寄存器是INA219实现直接读取电流/功率的核心，必须正确编程才能输出有效值。

### 8.1 编程步骤
1. 确定系统最大预期电流 $I_{max}$（单位：A）
2. 计算最小可实现电流LSB：
$$Current\_LSB_{min} = \frac{I_{max}}{32768}$$
（$2^{15}=32768$，16位有符号整数的有效值位数）
3. 选择方便计算的$Current\_LSB$：向上取整到易计算的数值（如100μA/bit），便于读取时直接换算
4. 计算校准值：
$$Cal = trunc( \frac{0.04096}{Current\_LSB \times R_{SHUNT}} )$$
（$R_{SHUNT}$为分流电阻阻值，单位Ω；$Current\_LSB$单位A/bit；`trunc`为取整函数）
5. 将计算得到的`Cal`值写入校准寄存器（16位无符号整数，bit0始终为0）
6. 写入配置寄存器（设置PGA、量程、工作模式等）

### 8.2 内部计算公式
- 电流值：$CURRENT = (Shunt\_Voltage\_Reg \times Cal) / 4096$ （自动加载到电流寄存器）
- 功率值：$POWER = (Current\_Reg \times Bus\_Voltage\_Reg) / 5000$
