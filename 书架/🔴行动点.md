# 🟥行动点

## 一、心态篇

### 1.1 基本态度

- **允许：尽人事而听天命**
- 经常感恩自己所拥有和经历过的一切——不论悲欢离合、得失成败，都让人生增添了一笔浓厚的色彩

> [!NOTE] 身边一切都值得感谢：感谢又活过一天，甚至一口饭菜都与无数人有着千丝万缕的联系。感谢每个愿意和我说话的人——在有限且宝贵的人生内愿意给予我关注

### 1.2 共情与付出

- 尽力共情背后的情绪、感受和需求，不要评判他人说话的方式
- **不要奢求回报**——起死回生的人都不会感恩上帝，你还要奢求什么？
- 人的本性就是忘恩负义，我们自身都常常无法做到知恩图报，又如何要求别人？

### 1.3 应对焦虑三步法

- 问自己三个问题：
    1. **我现在有什么呢？**——用自己有的去换自己想要的
    2. **三五年后会给我带来什么好处/坏处？**——如果你在生气，你是在对抗你自己
    3. **看清机制，防止盲目努力；找寻意义，注入长久动力；感受好处，保持愉悦**

### 1.4 对待他人的态度

- 允许讨厌我的人存在，无法决定他人的想法，做自己该做的就好
- 无需太过关注别人对你的态度——每个人都是自由的
- 如果别人给予赞赏或批评，应该由衷高兴：感谢别人在百忙之中抽出时间给我关注
- 做自己想做的事，无需理会别人的眼光，那与我无关

> [!NOTE] 并非事事都需要我们承担——大脑总是幻想不存在的"如果"，如果后台都在运行这些程序，我们可无法正常生活

- 无需在意他人的评论，强者才没时间管你

## 二、行动篇

### 2.1 任务管理

- **少量多餐**：把任务分阶段完成（拉伸区）
    - 大任务不要想着一次性完成，合理划分成小任务，逐个击破
- **少即是多**：我们能做的很少
    - 一次只通过一颗沙粒，一次只做好一件事情——《人性的优点》
    - 读一本书，只要有一个点可以让你的生活改变，那已经非常值得了——《认知觉醒》
- **要事先行**：记住怎么用石头和沙子装满空瓶——先放较大的石头，小沙粒才能填满剩余空间
    - 专注于重要关键的事情，合理分配才能避免琐事缠身、身心俱疲却一事无成
- **没有人一开始就能做好**：多给自己一点耐心
    - 多对自己说正指向的话，多给自己点鼓励
    - 面对困难，不要过早退出，对自己说"没关系的，我只会越做越好"

### 2.2 放松与活力

- 该使劲时使劲，该放松时放松——时时刻刻全身紧绷只会不知不觉中快速产生疲劳
- **精神放松是最高优先级**：只有精神放松了，肉体才会自然而然地放松

> [!warning] 长期紧张的危害：交感神经持续活跃 → 肌肉收紧、血流量增加、心率提高 → 血管发炎 → 血栓 → 高血压和心脏病

- 紧张的源头：
    1. 欲望太多，超越自身能力所及
    2. 情绪太多，内心中自导自演了无数个情绪戏剧

#### 放松学习，效率加倍

- 从反面讲：持续紧张状态下，感官灵敏度、专注程度、体能、精力都会下降
- 竞技选手的例子：顶级选手可以在得分后16~20秒内让心跳降到每分钟20次，而普通选手整场心率都停留在较高水平

#### 如何放松

- **生理层面**：冥想、紧绷后放松某个僵硬部位、深度呼吸（降低心率、积蓄能量）
- **心理层面**：
    1. **降低期待**：默认自己的结果就是不好，不要过高期望——跳进自己画的泥潭，只会越陷越深
    2. **写下来**：不要让模糊侵蚀自身，写下来、真实它、挖掘它、想办法解决它
    3. **专注目标**：我们能做的很少，能抛弃的就抛弃
    4. **身体放松**：觉得累了就平躺在地板上，伸展四肢，还可以打打滚，每天做两次

### 2.3 控制怒火

> [!warning] 面对已经发生的事情，时间不可逆转，"懊悔、抱怨、生气、争吵"毫无意义

- 林肯面对妻子直到临终前都担起了责任，眼睁睁看着战场上的机会被将军浪费，依然克制住了心中的怒火
- 思考对策化——坏事其实都是好事，上天在告诉我们应该如何更好的生活
- 如果命运给了我们一个柠檬，那就设法把它做成可口的柠檬吧

### 2.4 允许疑惑，继续前进

- 面对无法反驳的道理，做就好了，现实会告诉你答案
- 没有用脚丈量过的土地，你毫无概念——选择可以在实践中得到验证，慢慢纠正，总会形成可行的认知体系
- 允许疑问、质疑，但**不要停下脚步**
