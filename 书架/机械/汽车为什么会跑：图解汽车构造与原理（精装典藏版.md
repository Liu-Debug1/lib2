# 《汽车为什么会跑：图解汽车构造与原理》

## 一、发动机基本原理

### 1.1 动力来源

- 一切动力都来自气缸内的燃料燃烧推动活塞运动
- 动力传递路径：气缸 → 活塞 → 曲轴 → 连杆 → 变速器 → 传动轴 → 差速器 → 半轴 → 车轮

### 1.2 气缸数目

- 在同样功率要求下，缸数越多，缸径越小、转速越高、运转平衡性越好
- 但随着气缸数增加，零部件成比例增加，导致结构更复杂、可靠性降低、重量增加、成本上升

> [!NOTE] 随着增压技术和燃油喷射技术的进步，现在汽车发动机的气缸数有减少的趋势

## 二、发动机放置方式

| 放置方式 | 特点 |
|---------|------|
| **横置** | 发动机平行于车桥，一般用于前驱，为乘坐室留更多空间 |
| **纵置** | 发动机垂直于车桥 |
| **前置前驱** | 转向灵敏 |
| **后置后驱** | 加速快，转向过度沉，易漂移，不好操控 |

## 三、气缸排列形式

- **直列**：最常见的布局
- **水平对置**：运作时像两个拳击手你来我往，有效抵消震动，对称结构提高行驶稳定性。缺点是维修不便，排气声音怪异。目前只有保时捷、斯巴鲁和丰田等使用
- **V形**：减小发动机长度和高度，便于布置，降低发动机舱盖
- **W形**：V形基础上气缸15°错开，更短更轻，但宽度更大，震动较大
- **内燃机**：燃料在气缸内燃烧
- **外燃机**：早期蒸汽机，燃料在缸外燃烧烧开锅炉水，高压蒸汽推动活塞做功

## 四、发动机工作过程

### 4.1 动力与排量

- 汽车动力源自柴油或汽油燃烧产生的爆炸力
- **排气量** = 各气缸排量总和（下止点到上止点扫过的气体容积），单位cc/mL/L
- 排量越大，每次吸入可燃混合气越多，燃烧时产生的动力越强

### 4.2 燃烧原理

- 汽油中最主要的成分是碳氢化合物（只含碳和氢两种原子）
- 吸入的氧与碳氢化合物反应，产生水、CO、CO₂、NO和NO₂等

> [!NOTE] 同等重量下，汽油所含能量大约是糖的3倍、木头的5倍、**电池的200倍**。用电池取代汽油还需要电池技术研发人员加倍努力

### 4.3 四冲程工作循环

- 转速表反映的是发动机气缸内爆炸次数的体现——转速越高，爆炸次数越多，动力输出越大
- 转速6000rpm时：每秒100转 → 每活塞每秒50个工作循环 → 4缸发动机每秒200次爆炸

| 行程 | 进气门 | 排气门 | 活塞运动 | 说明 |
|------|--------|--------|---------|------|
| **进气** | 开 | 关 | 上止点→下止点 | 产生部分真空，吸入空气和汽油混合气 |
| **压缩** | 关 | 关 | 下止点→上止点 | 压缩混合气使其均匀、升温，便于燃烧 |
| **做功** | 关 | 关 | 上止点→下止点 | 火花塞点燃混合气，爆发出压力推动活塞 |
| **排气** | 关 | 开 | 下止点→上止点 | 排出燃烧过的废气，经消声器降噪 |

- 汽车一般采用四冲程发动机（曲轴转720°完成一次动力输出），二冲程发动机（曲轴转360°）一般用于摩托车
