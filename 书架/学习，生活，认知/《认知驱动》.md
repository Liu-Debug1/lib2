# 《认知驱动》

## 一、驱动力

### 1.1 明确动机

- 做每件事的动机会形成一种内在的驱动力和力量
- 做每件事情都不停问自己：有什么好处？带来什么影响？
- 在痛苦中寻找喜悦，在喜悦中迎接痛苦

> [!NOTE] 明确好处、消除模糊，才可以移开压在内心中不安、焦虑、迷茫的大山，让我们走得轻快

## 二、深度练习

- **学习即练**：心态上不追求急功近利，动作上尽量让自己快起来——是拉伸程度的快

## 三、人生目标

### 3.1 A计划：人生主线

- 找一个最符合自己的假设，结合现实不断调整，找到人生目标
- 标准的四个维度：
    1. **有价值**——在3年、5年、10年甚至更久的时间里都有用
    2. **能利他**——能解决自己和他人的痛点或给世界带来美，受益群体越大越好
    3. **可复制**——复制属性可以获取大量正反馈，拥有人生的无限可能
    4. **要跨界**——通过与其他技能复合，获取独特优势

- 我想做一个什么样的人：给别人带来幸福的人、汽车人、工程师、音乐家、合格的哥哥、合格的丈夫、合格的父亲
    - 为什么：人生有限，想凭借自己的一份力量为世界、为他人带来一份美好，这份美好在十年甚至百年都有重要价值

### 3.2 B计划：第二曲线

- **贝斯**——孤独前行，秘密行动
- 游走在圈内圈外的交错地带，避免不必要的压力与焦虑，保持愉悦

### 3.3 刚需：人生道路的必需品

- 阅读、早起、运动、冥想、反思

### 3.4 周期：七年的蜕变

- 蜕变的周期可能是三年、五年、七年——总之肯定不是几十天
- 给自己多一点耐心去改变
- 方法：刻意练习 + 与时间做朋友 + 打造出尽自己所能最好的产品

## 四、远离二元对立的世界

- 人们习惯将问题简单化、单一化、非黑即白——这样大脑不需要消耗能量去思考
- 心灵得到淬炼时会发现，一切事件远没有想象那么简单。很多人明白却趋于逃避，成功之人永远敢于直面痛苦
- **直面困惑、消除困惑是一生的课题**

> [!warning] 逃避只是一时的，如果不去解决，它永远都在那里。等到必要之时，它会露出凶猛的獠牙向你猛扑
