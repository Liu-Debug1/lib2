# 《认知天性》

## 一、给学生的学习策略

### 1.1 检索练习：合上书自测

- 时不时合上课本，问自己：核心概念是什么？哪些术语没接触过？如何定义它们？这些概念和我已知的东西有什么联系？
- 每周留出时间自测
- **好处**：了解自己是否记住并强化、发现薄弱点、提高学习目的性和专注性、带动思考和调动记忆

> [!NOTE] 自测会带来生疏和沮丧的感觉，但就在你费力检索新知识时，每一次努力的回想实际上都是一个加深记忆的过程

### 1.2 有间隔地安排检索练习

- 不止一次地学习资料，两次练习中间要隔开一段时间
- 建立自测计划，每个学习阶段之间留出间隔——如思维导图（不要太详细，要让自己回忆）、练习题
- 复习不能只是阅读，要自测能否回忆起来
- 最低频率：每月1次

> [!NOTE] 有间隔的练习让人感觉更困难，但你在长期记忆中重建所学时，不仅强化了记忆，还强化了对知识的掌握

### 1.3 穿插安排不同类型的问题

- 先理解思路，但不要着急做大量例题——继续学新知识去
- 把前边剩下的例题分散到练习规划中，囤积多种类型后，有间隔、乱序地去解决
- **好处**：避免集中练习带来的认知偏差、珍惜练习资源、有效带动大脑检索能力

> 打断针对一种类型问题的练习去进行另一种类型的练习，感觉很突兀且没有成效。但研究证明这只是幻觉——明白这一点会使你获得优势

### 1.4 细化与生成

- **细化**：把资料和你已知的东西联系起来，用自己的语言向别人解释；为新资料提供比喻或视觉形象
- **生成**：在得到答案或解决方案前，尝试回答疑问或解决问题
- 课前一定要阅读资料

## 二、学习是挑战天性的必修课

| 低效方法 | 高效方法 |
|---------|---------|
| 填鸭式（重复、被动） | 检索练习（主动、回想、强化路径） |
| 集中练习 | 间隔练习（困难、生疏但更牢靠） |
| 单一类型练习 | 穿插学习（加强提炼共性、提高归纳能力） |
| 死记硬背 | 细化（不断理解、用已知解释未知） |

- 越是轻松、简单的学习越是低效——临时抱佛脚、反复阅读都是典型
- 扎实地学习一定伴随着痛苦和困难
- 重复式阅读往往给我们带来已经掌握的假象（元认知错觉）

> [!warning] 如果感到学习非常吃力，说明你正在学习非常重要的东西。犯错误并改正错误，其实是在搭建通往高层次学习的桥梁。
