# 《精力管理》

## 一、核心概念

- 精力管理不是时间管理，而是**能量管理**
- 精力来源的四个维度：体能、情感、思维、意志

## 二、体能精力

- 深呼吸：降低心率，积蓄能量
- 睡眠：保证充足高质量的睡眠
- 饮食：
    - **拒绝断食**——体重狂掉的大部分不是脂肪，而是水分、宝贵的蛋白质和糖分
    - 随之而来的是免疫力和精力的急剧下降、情绪低落、易怒、大脑昏沉
    - 减肥的本质不是体重秤上的数字变化，而是需要健康的、可长久坚持的、令人舒服的饮食习惯

> [!warning] 断食的后果：免疫力急剧下降、情绪低落易怒、提不起精神

### 饮食四要素

| 类别 | 原则 |
|------|------|
| **糖类** | 选择慢碳（粗粮、豆类），避免高升糖指数的精制碳水 |
| **蛋白质** | 体力活动少时每kg体重0.8~1.2g；运动人群1.2~1.8g（65kg约60g，一个鸡蛋含6~7g） |
| **脂肪** | 每kg体重1g以内，减脂0.8g以内。女性不低于0.6g。避免反式脂肪酸 |
| **水** | 每kg体重30ml（65kg约2L），小口多次，养成持续补水习惯 |
| **维生素** | 蔬菜所含的维生素比水果多得多 |

## 三、冥想

- 人在平静状态下，能量不会耗散
- 长期冥想训练的人，大脑前额叶皮质中的灰质增加——有可能获得更发达的前额叶皮质，从而更好地控制情绪和选择

### 方法1：盘腿静坐冥想法

1. 找一处安静不受干扰的地方，盘腿静坐，双手自然垂放在膝盖上
2. 闭上眼睛，把全身精力集中在呼吸上
3. 用腹部呼吸，深深吸气，腹部内收
4. 吸到最大，憋气
5. 缓缓呼气，腹部外松
6. 呼出全部，屏气凝神

> [!NOTE] 如果注意力不集中、大脑冒出其他想法，没关系——承认这个想法，再把它放走，意识始终关注于呼吸

### 方法2：数呼吸冥想法

- 吸气，想象美好气流缓慢地从鼻腔进入身体，带来舒适
- 吐气，想象不好的废气缓慢地从鼻腔离开身体，让身心净化
- 从1数到10，再重新开始
- 每天睡前进行5分钟冥想，卸掉一整天的压力
