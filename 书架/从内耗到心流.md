# 《从内耗到心流》

## 一、核心概念：习得性无助

- 如果经常达不到预期，人便会倾向于"接受不完美的自己"，放弃变得更好——这就是著名的**习得性无助**
- 习得性无助的四个阶段：
    1. 以偏概全，认知错误
    2. 放弃努力，拒绝尝试
    3. 自我否定，放大无力感
    4. 错误归因

> [!warning] 警惕恶性循环：预期失败 → 放弃尝试 → 验证失败 → 更深的无力感

## 二、心流状态

- 心流的本质：全身心投入一件事，忘记时间流逝，过程本身就是奖励
- 进入心流的条件：
    - 目标清晰可量化
    - 即时反馈（知道每一步做得如何）
    - 挑战与技能匹配（处于拉伸区，不过难也不过易）
