# 选择题设计原则

> 来源：NBME 命题指南、王后雄《教育考试的理论与方法》、雷新勇《试题命制的理论和技术》、黄光扬《教育测量与评价》

## 一、题干设计

| 原则 | 说明 | 反例 |
|------|------|------|
| 题干意义完整 | 遮住选项，题干本身应能明确告知需要解决的问题 | "以下说法正确的是？" ❌ |
| 一个中心问题 | 每道题只围绕一个知识点，不交叉考核 | 一道题同时考指针 + class + 宏定义 ❌ |
| 避免否定表述 | 不用"下列哪项不是..."，除非否定是考核要点 | 可少用但应加粗标注 **不是** |

## 二、选项设计（干扰项）

### 四大陷阱类型

| 类型 | 说明 | C/C++ 举例 |
|------|------|-----------|
| 概念混淆 | 与正确答案概念相近、容易混淆 | 问 `enum class` 特性，选项放私有/公有（混淆 class 访问控制） |
| 逻辑漏洞 | 看似合理但有逻辑缺陷 | 问宏括号作用，选项放"提高执行效率" |
| 数据干扰 | 数字/数值上迷惑 | 问 `sizeof(int*)`，选项放 1/2/4/8 各种常见值 |
| 情境迁移 | 从其他语境搬来的合理但不适用于本题的说法 | 问 C++ struct 特性，选项放 C struct 的规则 |

### 选项规范

- **同质性**：各选项长度、句式复杂度、术语应统一
- **防暗示**：正确选项不比其他选项明显更长或更短
- **不用模式化词**："总是/从来/所有" → 通常错误，"通常/可能/一般" → 通常正确
- **选项个数**：研究表明 3 个精心设计的选项即可达到 4 个选项同等的信度
- **答案分布**：正确答案在 A/B/C/D 各位置出现次数应大致均匀

## 三、多选题特殊规则

| 原则 | 说明 |
|------|------|
| 正确答案个数 | 2-3 个为宜（不超过总选项的一半） |
| 先行标注 | 题干标明"（多选）" |
| 部分得分 | 少选得部分分，错选 0 分 |
| 独立选项 | 每个选项互相独立，不存在"选A就必须选B"的依赖关系 |

## 四、选择题常见缺陷（NBME 研究：56% 题目至少含一种）

1. 选项长度不均 → 长选项倾向正确
2. 语法衔接线索 → "an" 暗示元音开头选项
3. 逻辑线索 → "以上都是"常为正确答案
4. 无关难度 → 用生僻术语增加并非考点本身的难度
5. 拼凑选项 → 凑数用的、任何人都不会选的选项

## 五、选择题出题六步法

1. 确定测量目标（考什么知识点、什么认知层次）
2. 撰写题干草稿
3. 确定正确答案
4. 设计干扰项（从真实错误经验中找）
5. 通读检查：选项同质、无暗示、无缺陷
6. 试答 + 磨题：命题人先做一遍，修改不合理的
